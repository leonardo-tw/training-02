About Mimic

Mimic is an interactive art tool built using p5.js, designed to explore image manipulation in real time. This project allows users to transform, distort, and remix images through various interactions, including mouse movements and keyboard shortcuts.

By dynamically cutting and rearranging visual elements, Mimic lets users create unique, abstract compositions. The project generates new designs based on user input and randomness, offering endless creative possibilities for generating posters.

The tool is currently in its alpha stage, with continuous updates to enhance the user experience and add more creative options. Explore the shortcuts, manipulate images, and generate custom visuals with a simple click.
